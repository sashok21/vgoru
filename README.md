# Сайт для гірських маршрутів

## Опис проєкту

"Гірські маршрути України" — це вебдодаток, створений для любителів гірського туризму та мандрівників. Платформа надає зручний доступ до каталогу популярних гірських маршрутів Українських Карпат з детальною інформацією про кожен з них: складність, тривалість, висоту, краєвиди та особливості. Користувачі можуть переглядати маршрути, шукати найцікавіші локації та планувати свої майбутні подорожі в гори.

## Мета проєкту

Мета проєкту полягає у створенні зручного онлайн-каталогу для мандрівників, який допоможе:
- Знайти найкращі гірські маршрути відповідно до рівня підготовки
- Отримати детальну інформацію про кожен маршрут перед походом
- Переглянути маршрути на інтерактивних картах
- Ділитися власним досвідом через відгуки та оцінки
- Зберігати історію пройдених маршрутів у особистому профілі

## Основні функції

- **Перегляд маршрутів**: повний каталог гірських маршрутів з фотографіями, детальним описом, інформацією про складність, висоту, тривалість та особливості кожного маршруту
- **Пошук і фільтрація**: можливість знаходити маршрути за складністю (легкий, середній, важкий), тривалістю походу або географічним регіоном
- **Інтерактивні карти**: перегляд детальної інформації про маршрут на картах з відміткою ключових точок та висотного профілю
- **Відгуки та оцінки**: зареєстровані користувачі можуть залишати свої враження, рекомендації та оцінки маршрутів
- **Особистий профіль**: кожен користувач має доступ до власного профілю з історією пройдених маршрутів, списком бажаних локацій та статистикою

## Використані технології

- **Backend**: Django (Python) — потужний веб-фреймворк для швидкої розробки
- **Frontend**: HTML5, CSS3, JavaScript — сучасні технології для створення адаптивного інтерфейсу
- **База даних**: SQLite (для розробки) / PostgreSQL (для продакшену)
- **Картографічний сервіс**: інтеграція з Google Maps API або OpenStreetMap для відображення маршрутів
- **Додатково**: Django ORM, Django Template Engine, CSS Grid/Flexbox для адаптивного дизайну

## Інструкція з запуску

### 1. Клонувати репозиторій

```bash
git clone <посилання-на-репозиторій>
```

### 2. Перейти в директорію проєкту

```bash
cd vgoru
```

### 3. Створити та активувати віртуальне середовище

**Для Linux/MacOS:**
```bash
python3 -m venv venv
source venv/bin/activate
```

**Для Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

### 4. Встановити залежності

```bash
pip install -r requirements.txt
```

### 5. Виконати міграції бази даних

```bash
python manage.py migrate
```

### 6. (Опціонально) Створити суперкористувача для адмін-панелі

```bash
python manage.py createsuperuser
```

### 7. Запустити сервер розробки

```bash
python manage.py runserver
```

Після цього відкрийте браузер та перейдіть за адресою: `http://127.0.0.1:8000/`

Адмін-панель доступна за адресою: `http://127.0.0.1:8000/admin/`
